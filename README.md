**About this repository**

This repository is a *laboratory assignment* for my university course, focused on practicing and understanding **Continuous Integration** and **Continuous Deployment (CI/CD)** concepts.

Key features:
- Hands-on experience with automating software workflows using **GitHub Actions**
- Practical exercises and configurations for learning modern DevOps practices

These materials are designed to help students gain real-world skills in CI/CD and DevOps automation.
